Box
========================


Import the required mbuild package.

.. code:: ipython3

    import mbuild as mb



Build an empty orthogonal **mBuild** Box measuring 4.0 nm in all the x, y, and z-dimensions.   

.. code:: ipython3

    empty_box = mb.load(lengths=[4.0, 4.0, 4.0], angles=[90, 90, 90])
